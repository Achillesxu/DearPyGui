Dear PyGui

Item Configuration

In Dear PyGui, when GUI items are added, various configuration options and flags can be set. There are several options common to all items (i.e. show) but most GUI items have specific options. In order to modify an item's configuration after being created, you can use the configure_item(...) command in conjuction with the keyword from the item's add_* command. You can also retrieve an item's configuration in the form of a dictionary by using the get_item_configuration(..) command.

Example

Simple usage can be found below: 

add_button("Button 1", enabled=True, label="Press me")

# at a later time, change the item's configuration
configure_item("Button 1", enabled=False, label="New Label")
