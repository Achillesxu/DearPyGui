Dear PyGui

These constants exist inside of the dearpygui.core module

Constants are recommended because they are all cross-platform in Dear PyGui

Plot Colors


            mvPlotCol_Line          // plot line/outline color (defaults to a rotating color set)
            mvPlotCol_Fill          // plot fill color for bars (defaults to the current line color)
            mvPlotCol_MarkerOutline // marker outline color (defaults to the current line color)
            mvPlotCol_MarkerFill    // marker fill color (defaults to the current line color)
            mvPlotCol_ErrorBar      // error bar color (defaults to ImGuiCol_Text)
            mvPlotCol_FrameBg       // plot frame background color (defaults to ImGuiCol_FrameBg)
            mvPlotCol_PlotBg        // plot area background color (defaults to ImGuiCol_WindowBg)
            mvPlotCol_PlotBorder    // plot area border color (defaults to ImGuiCol_Text)
            mvPlotCol_XAxis         // x-axis grid/label color (defaults to 25% ImGuiCol_Text)
            mvPlotCol_YAxis         // y-axis grid/label color (defaults to 25% ImGuiCol_Text)
            mvPlotCol_YAxis2        // 2nd y-axis grid/label color (defaults to 25% ImGuiCol_Text)
            mvPlotCol_YAxis3        // 3rd y-axis grid/label color (defaults to 25% ImGuiCol_Text)
            mvPlotCol_Selection     // box-selection color (defaults to yellow)
            mvPlotCol_Query         // box-query color (defaults to green)
            mvPlotCol_COUNT

Plot Marker Specifications


			mvPlotMarker_None      // no marker
			mvPlotMarker_Circle    // a circle marker will be rendered at each point
			mvPlotMarker_Square    // a square maker will be rendered at each point
			mvPlotMarker_Diamond   // a diamond marker will be rendered at each point
			mvPlotMarker_Up        // an upward-pointing triangle marker will up rendered at each point
			mvPlotMarker_Down      // an downward-pointing triangle marker will up rendered at each point
			mvPlotMarker_Left      // an leftward-pointing triangle marker will up rendered at each point
			mvPlotMarker_Right     // an rightward-pointing triangle marker will up rendered at each point
			mvPlotMarker_Cross     // a cross marker will be rendered at each point (not filled)
			mvPlotMarker_Plus      // a plus marker will be rendered at each point (not filled)
			mvPlotMarker_Asterisk  // a asterisk marker will be rendered at each point (not filled)
